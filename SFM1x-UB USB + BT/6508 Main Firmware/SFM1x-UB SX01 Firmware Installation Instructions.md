Firmware Installation Instructions for SFM1x-UB with serial numbers starting with "SX01" only.
PLEASE NOTE, this firmware is NOT compatible with SFMx serial numbers starting "SP01", installing it on these devices will render
the device inoperable.

1)  Disconnect external power such as Solar Panel connected to the BusPlugs
2)  Connect USB cable, open and connect SFM with the ICT Combined Instrument Software
3)  Change the logging setting to "Manual" and update the changes, if a measurement is running, 
	wait for it to finish before the next step.
4)  Disconnect USB cable
5)  Power off the SFM by the power switch procedure.
6)  Eject the SD card.
7)  Place the SD card into a Windows 10 or Windows 11 PC
8)	It is advised to rename or backup and remove the original data files from the SD Card
	This ensures the SFM can create new files after firmware update.
9)  Download the firmware file, and copy the firmware file to the SD card, e.g. autofw-R1-1-15_09022024.sfm
10) Rename the file such that is name is autofw, and the extension remains as .sfm
11) Eject the SD card from the PC.
12) Insert SD card into the SFM.
13) Power on the SFM by holding the power button for 5seconds, then release.
14) The Green/Red LED in light pipe will begin the rapidly flash, then will cease and flash slowly green to indicate a reboot after successful update.
15) Repeat step 5) above.
16) Repeat step 6) above.
17) Repeat step 7) above.
18) For a successful firmware install the file is renamed automatically to update.sfm
19) Delete the file update.sfm.
20) Firmware update is complete.
21) Power on the device, connect original external power.
22) Connect USB cable, then check local time is correct on the SFM.
23) Set desired logging interval, check data after a measurement is complete.
24) Remove USB cable, and install USB Port Cap to ensure the SFM is Dust and Waterproof.
